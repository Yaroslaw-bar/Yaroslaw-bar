## Проекты

### Elkadzor Review Monitor

> Автоматическая система мониторинга отзывов о компании «Элькадор» — производителе заборов и ворот в Санкт-Петербурге.

Пет-проект, который я сделал в рамках обучения работе с Claude Code. Система собирает отзывы с нескольких площадок, сохраняет их в базу данных, анализирует тональность и показывает результат в веб-дашборде.

**Зачем:** отзывы разбросаны по разным сайтам, за ними неудобно следить вручную. Проект собирает всё в одно место, выделяет новые отзывы и помогает быстро понять, положительные они или негативные.

**Возможности:**
- Сбор отзывов с сайта `elcador.ru/otzyvy` и Яндекс.Карт
- Дедупликация: один отзыв не попадает в базу дважды
- Автоматический анализ тональности: положительный / нейтральный / отрицательный
- Telegram-уведомления о новых отзывах
- Веб-дашборд на Streamlit с фильтрами, сортировкой и экспортом в CSV
- Планировщик для регулярного мониторинга

**Стек:** `Python`, `requests`, `BeautifulSoup`, `Playwright`, `SQLite`, `Streamlit`, `python-telegram-bot`, `APScheduler`, `python-dotenv`

**Ключевые файлы:**
- `parser.py` — статические парсеры
- `dynamic_parser.py` — парсеры через браузер
- `db.py` — работа с SQLite
- `dashboard.py` — интерфейс на Streamlit
- `collect_reviews.py` — быстрый запуск сбора отзывов
- `scheduler.py` — фоновый мониторинг

**Статус площадок:**
| Площадка | Статус | Примечание |
|---|---|---|
| Сайт `elcador.ru/otzyvy` | Работает | 279 отзывов, средний рейтинг 4.84 |
| Яндекс.Карты | Работает | 20 отзывов, средний рейтинг 4.8 |
| 2GIS | Капча | Требует антибот-обхода |
| Plaso.pro | Капча | Блокируется проверкой |
| RemKarta | Не загружается в HTML | Требует доработки |
| Zoon.ru | Нет текстовых отзывов | Только оценки |

**Чему научился:**
- Парсить статичные и динамичные сайты
- Работать с Playwright и обходить JavaScript-рендеринг
- Нормализовать даты из разных форматов
- Делать лексический анализ тональности на русском языке
- Дедуплицировать данные
- Подключать уведомления, планировщик и веб-интерфейс
