# Elkadzor Review Monitor

> Автоматическая система мониторинга отзывов о компании «Элькадор» — производителе заборов и ворот в Санкт-Петербурге.

## Что это

Пет-проект, который я сделал в рамках обучения работе с Claude Code. Система собирает отзывы с нескольких площадок, сохраняет их в базу данных, анализирует тональность и показывает результат в веб-дашборде.

## Зачем

Отзывы о компании разбросаны по разным сайтам. Вручную следить за ними неудобно. Этот проект решает задачу: собрать всё в одно место, выделить новые отзывы и быстро понять, положительные они или негативные.

## Возможности

- **Сбор отзывов** с сайта `elcador.ru/otzyvy` и Яндекс.Карт
- **Дедупликация** — один и тот же отзыв не попадает в базу дважды
- **Анализ тональности** — автоматическая метка: положительный / нейтральный / отрицательный
- **Telegram-уведомления** о новых отзывах
- **Веб-дашборд** на Streamlit с фильтрами, сортировкой и экспортом в CSV
- **Планировщик** для регулярного мониторинга

## Стек

| Назначение | Инструмент |
|------------|-----------|
| Статический парсинг | `requests`, `BeautifulSoup`, `lxml` |
| Динамический парсинг | `Playwright` |
| Хранение данных | `SQLite` |
| Дашборд | `Streamlit` |
| Уведомления | `python-telegram-bot` + Telegram Bot API |
| Планировщик | `APScheduler` |
| Конфигурация | `python-dotenv` |

## Архитектура

```
config.py          — источники, URL, переменные окружения
parser.py          — статические парсеры
 dynamic_parser.py  — парсеры через браузер (Playwright)
db.py              — работа с SQLite
notifier.py        — отправка уведомлений в Telegram
scheduler.py       — периодический запуск сбора
collect_reviews.py — быстрое обновление активных площадок
dashboard.py       — веб-интерфейс на Streamlit
```

## Как работает сбор

1. `collect_reviews.py` запускает парсеры для активных площадок.
2. Каждый отзыв проверяется на дубликат по сочетанию `source + text`.
3. Новые записи сохраняются в `reviews.db`.
4. Если настроен Telegram, приходит уведомление с кратким списком новых отзывов.

## Дашборд

Дашборд показывает:

- Статистику по площадкам: количество отзывов и средний рейтинг
- Список отзывов с фильтрами по площадке и диапазону дат
- Сортировку по дате, оценке и тональности
- Развёрнутые карточки отзывов с текстом, оценкой звёздами, меткой тональности и рекомендацией по ответу
- Кнопку экспорта в CSV

## Запуск

```powershell
cd C:\project\elkadzor-reviews
python -m venv .venv
.\.venv\Scripts\pip install -r requirements.txt
.\.venv\Scripts\playwright install chromium
copy .env.example .env
# добавь TELEGRAM_BOT_TOKEN и TELEGRAM_CHAT_ID

# сбор отзывов
.\.venv\Scripts\python collect_reviews.py

# дашборд
$env:STREAMLIT_SERVER_HEADLESS="true"
.\.venv\Scripts\python -m streamlit run dashboard.py
```

Дашборд откроется на `http://localhost:8501`.
